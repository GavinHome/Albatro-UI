# albatro-theme
> albatro component theme.


## Installation
```shell
npm i albatro-theme -S
```

## Usage

Use Sass import
```css
@import 'albatro-theme';
```

Or Use webpack
```javascript
import 'albatro-theme';
```

Or
```html
<link rel="stylesheet" href="path/to/node_modules/albatro-theme/lib/index.css">
```

##  Import on demand
```javascript
import 'albatro-theme/lib/button.css';

// ...
```
